# Instagram Reels Videos

This folder should contain the video files and creator profile pictures for the Instagram reels section.

## Required Files

### Videos (5 files):
- `reel1.mp4` - First reel video
- `reel2.mp4` - Second reel video
- `reel3.mp4` - Third reel video
- `reel4.mp4` - Fourth reel video
- `reel5.mp4` - Fifth reel video

### Profile Pictures (5 files):
- `creator1.jpg` - Profile picture for first reel
- `creator2.jpg` - Profile picture for second reel
- `creator3.jpg` - Profile picture for third reel
- `creator4.jpg` - Profile picture for fourth reel
- `creator5.jpg` - Profile picture for fifth reel

## Video Specifications

- **Format**: MP4 (H.264 codec recommended)
- **Aspect Ratio**: 9:16 (vertical/portrait orientation - Instagram Reels format)
- **Recommended Resolution**: 1080x1920 pixels
- **File Size**: Keep under 10MB per video for optimal loading
- **Duration**: 15-60 seconds recommended

## Profile Picture Specifications

- **Format**: JPG or PNG
- **Aspect Ratio**: 1:1 (square)
- **Recommended Size**: 400x400 pixels minimum
- **File Size**: Keep under 500KB per image

## Notes

- All videos should be optimized for web playback
- Videos will be muted by default
- Videos will loop automatically when playing
- If profile pictures fail to load, the site logo will be used as a fallback
